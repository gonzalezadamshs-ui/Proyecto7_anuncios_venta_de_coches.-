# Proyecto7_anuncios_venta_de_coches.-

Repositorio para crear aplicación de anuncios de venta de coches.
🚗 Dashboard Interactivo de Venta de Vehículos
📌 Descripción

Este proyecto consiste en el desarrollo de una aplicación web interactiva utilizando Python, Streamlit y Plotly, diseñada para explorar y analizar un conjunto de datos sobre anuncios de venta de vehículos usados.

El objetivo principal es ofrecer una interfaz intuitiva que permita filtrar información, visualizar estadísticas y comprender mejor las tendencias del mercado automotriz mediante gráficos dinámicos e interactivos.
